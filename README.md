# 📹 Campanha de Acesso às Câmeras

Landing page desenvolvida para conscientizar e orientar clientes sobre a importância de fornecer acesso às imagens de suas câmeras de segurança para a central de monitoramento.

A campanha tem como objetivo agilizar a verificação de ocorrências e aumentar a precisão na identificação de invasões, disparos de alarme e outras situações emergenciais.

Ao disponibilizar acesso às imagens, a equipe consegue analisar eventos em tempo real e tomar decisões mais rápidas e assertivas.

## 🎯 Objetivo do Projeto

O objetivo desta campanha é conscientizar os clientes sobre a importância do acesso às imagens das câmeras para tornar o monitoramento mais eficiente.

Com isso:

✔️ A equipe consegue verificar eventos mais rapidamente  
✔️ A análise se torna mais precisa  
✔️ Reduz o tempo de resposta em ocorrências  
✔️ Aumenta a segurança do cliente
